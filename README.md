# Beaver Valley — Official Site

Marketing / investor + gamer website for **Beaver Valley**, a grimy dystopian-satire CRPG × physics beat-em-up hybrid where ~250 characters each have a mind of their own.

A one-developer game by **Jeremy Thorn**.

## Pages
- `index.html` — What Beaver Valley is (pitch, hooks, modes, live music & crossover cast)
- `features.html` — What's already built (with real in-engine screenshots)
- `world.html` — The world, block by block (location gallery)
- `roadmap.html` — In Progress / Planned / Vision
- `donate.html` — Support the game (Streamlabs tips → credits / custom character)

## Running it
Static site — no build step. Either open `index.html` directly, or serve it:

```bash
python -m http.server 8777
# then visit http://localhost:8777
```

## Stack
Plain HTML/CSS/JS. Theme derived from the game's main-menu key art. Fonts via Google Fonts with system fallbacks. All imagery is real game art / in-engine captures.
